**Internal utility package for nx-extend plugins.**

This package contains shared utilities and helper functions used across all @nx-extend plugins. It is not intended for direct installation or use by end users.

## Usage

This package is automatically installed as a dependency when you install any @nx-extend plugin. You do not need to install it separately.
